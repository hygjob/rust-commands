# Rust를 이용한 커맨드 툴 제작 — 목차

| 장   | 커맨드                | 핵심 주제                                                    |
| ---- | --------------------- | ------------------------------------------------------------ |
| 1    | **echo**              | 개발 환경 설정, `clap`으로 인자 파싱, 표준 출력, 첫 CLI 빌드 |
| 2    | **cat**               | 파일 I/O (`File`, `BufReader`), `Result`와 `?` 연산자, 에러 처리 |
| 3    | **head / tail**       | 버퍼링, `Seek` 트레이트, 옵션 플래그 설계 (`-n`, `-c`)       |
| 4    | **grep**              | `regex` 크레이트, 패턴 매칭, 컬러 출력, 라인 컨텍스트        |
| 5    | **sort**              | `Ord` 트레이트, 비교 함수, 숫자/역순/대소문자 정렬           |
| 6    | **cut / tr**          | 문자열 분할, 필드 추출, 문자 매핑, 이터레이터 체이닝         |
| 7    | **find**              | `walkdir` 크레이트, 재귀 순회, 이름/타입/크기 필터           |
| 8    | **ls**                | 디렉토리 읽기, 파일 메타데이터, 권한 문자열, `-l` 긴 형식    |
| 9    | **tree**              | 재귀 트리 출력, 접두어 시각화, 디렉토리/파일 카운트          |
| 10   | **cp / mv**           | 파일 복사·이동, 덮어쓰기 확인, 진행률 표시                   |
| 11   | **du**                | 재귀 용량 합산, 사람 친화적 포맷(`KB/MB/GB`), 심볼릭 링크    |
| 12   | **xargs**             | `std::process::Command`, 서브프로세스 실행, 인자 분할, 파이프 |
| 13   | **터미널 인터페이스** | `colored`, `indicatif` 진행률 바, `comfy-table` 테이블 출력  |
| 14   | **테스트와 배포**     | `assert_cmd` 통합 테스트, `cargo install`, GitHub Actions    |
| 15   | **통합 멀티툴**       | workspace 구성, 하나의 바이너리에 12개 커맨드 통합           |
